# DocuMind

DocuMind is a RAG-based enterprise document Q&A system.

Users can upload PDF documents, and the system will parse the content, split it into chunks, generate embeddings, retrieve relevant context, and generate source-grounded AI answers with citations.

## Tech Stack

### Frontend

- Vue 3
- TypeScript
- Vite
- Ant Design Vue

### Backend

- Python
- FastAPI
- PyMuPDF
- Vector Database
- LLM API

## Core Features

- PDF upload
- Document parsing
- Text chunking
- Embedding generation
- Vector search
- AI question answering
- Source citations
